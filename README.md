Superstore Sales Dashboard
Overview

This project was completed as part of my Data Analyst Internship (Task 3). The objective was to design an interactive dashboard in Power BI that enables business users to analyze sales performance, profitability, and business trends through meaningful visualizations.

Objective

The goal of this project was to:

Design an interactive business dashboard.
Visualize key business metrics using Power BI.
Perform sales and profit analysis.
Support decision-making through interactive reports.
Tools Used
Power BI
Microsoft Excel
Dataset

Superstore Sales Dataset

Dashboard Features
KPI Cards for Total Sales, Total Profit, Total Orders, Total Quantity, and Profit Margin.
Sales by Category using a Bar Chart.
Profit by Region using a Column Chart.
Sales Trend Over Time using a Line Chart.
Sales by Segment using a Donut Chart.
Sales by Country using a Map.
Sales by Sub-Category using a Treemap.
Interactive slicers for Year, Region, and Category.
Key Insights
Compared sales performance across product categories.
Analyzed regional profit distribution.
Identified sales trends over time.
Examined sales contribution by customer segment.
Enabled interactive filtering for detailed business analysis.
